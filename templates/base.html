Simulation Mode

This platform is running in simulation mode. This means:

  • No real money is transferred
  • EcoCash confirmations are simulated
  • Binance API calls are simulated
  • Exchange rates are simulated (near real)

To go live, the admin needs to:

  1. Add real Binance API keys
  2. Set up EcoCash Merchant account
  3. Set SIMULATION_MODE = False in settings
